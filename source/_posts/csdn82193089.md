## 题目描述

*   给定一个double类型的浮点数base和int类型的整数exponent。
*   求base的exponent次方。

## 解题思路

**思路1**

```
class Solution:
    def Power(self, base, exponent):
        return base**exponent
```

**思路2**

```
class Solution:
    def Power(self, base, exponent):
        if base == 0:
            return 0
        exp = abs(exponent)
        result = 1.0
        for i in range(exp):
            result *= base
        if exponent > 0:
            return result
        else:
            return 1/result
```
