## 反向代理

可以在反向代理后面设置气流，并能够灵活地设置其端点。

例如，您可以配置反向代理以获取：

```
 https : // lab . mycompany . com / myorg / airflow / 
```

为此，您需要在<cite>airflow.cfg中</cite>设置以下设置：

```
 base_url = http : // my_host / myorg / airflow 
```
